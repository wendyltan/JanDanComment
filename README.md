### 段子区介绍：

![段子区界面][6]
#### 主要ui功能：
- 显示当前页数。这个页数是对应网页页数的。也就是网页版内容当前段子的而最新页号
- 跳转页数。输入页数可以直接跳转。
- 刷新：
	+ **上拉**：跳到上一页
	+ **下拉**：跳到下一页
- 吐槽：点击之后可以查看相应段子的评论内容
- menu：右下角三个点，用于分享段子。
